21、v-charts的options

发表于 2020-04-28 更新于 2022-09-05

v-charts 的 options 使用 getOptions()而不是使用一个变量

使用 getOptions()是为了获取数据之后才进行渲染数据，v-chart 的 options 可以直接绑定一个变量,在请求数据的时候，更新变量，图表需要进行重新渲染

这里的 getOptions()也可以写在 computed 里面的。
