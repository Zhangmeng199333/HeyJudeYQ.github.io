55、大文件断点续传

2022-02-21

关于大文件断点续传的问题,解决方案是Blob.prototype.slice方法，和数组的slice方法类似，使用slice方法可以返回源文件的切片。按照要求将源文件切位n个切片，将多个切片同时上传，源文件由一个大文件转换成n个小切片同时上传，可以大大减少上传时间。需要注意的是上传到服务端的切片可能位置会发生改变，需要将切片的位置记录下来。
