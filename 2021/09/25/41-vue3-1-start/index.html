三、组合式 API：composition API

composition api 为 Vue 应用提供更好的逻辑复用和代码组织。

setup()钩子是在组件中使用组合式 API 的入口，通常只在以下情况下使用：

1、需要在非单文件组件中使用组合式 API 时。

2、需要在基于选项式 API 的组件中集成基于组合式 API 的代码时。
