46、解决GitHub Pages绑定阿里域名CNAME消失

2021-10-28

hexo主题使用的是Claudia,部署到GitHub Pages时，CNAME会消失。

解决方式：

// 1 cd source
touch CNAME

// 2 输入阿里域名
issummer.cn

// 3 部署
hexo g && hexo d
