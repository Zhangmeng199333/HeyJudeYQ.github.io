hexo常用命令

Quick Start

新建文件

$ hexo new "My New Post"

More info: Writing

启动本地服务

$ hexo server

More info: Server

生成本地静态文件

$ hexo generate

More info: Generating

发布

$ hexo deploy

More info: Deployment

打包发布的命令行可以直接这样写：

hexo g && hexo d
