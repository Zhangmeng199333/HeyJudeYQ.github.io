12、下拉框select银行卡号格式处理

2019-12-29

银行卡号显示为 6217 **** 0000 的格式

以选择银行卡为例：

    <select v-model="selected" class="selected">
        <option disabled value v-show="showBankCard">请选择提现到账的银行卡</option>
        <option
            v-for="item in BankCardList"
            :key="item.id"
            v-bind:value="item.id"
        >
            {{ item.card_no }}
        </option>
    </select>

item.card_no ? item.card_no.replace(/^(\d{4})\d+(\d{4})$/,"$1****$2"):""
// 银行卡号显示为 6217 **** 0000 的格式。
