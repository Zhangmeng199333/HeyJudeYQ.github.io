106、ref小技巧

2022-12-12

使用 console.log 输出一个 ref 对象时，控制台查看结果时很不方便，Chrome 浏览器对此有一个格式化的处理。

示例：

<script setup lang="ts">
import { ref } from 'vue'
const str = ref<string>('jude')
console.log('str',str)
</script>

控制台打印结果为

RefImp{
  dep:undefined
  __v_isRef:true
  __v_isShallow:false
  _rawValue:'jude'
  _value:'jude'
  value:'jude'
}

点击控制台右上角设置按钮 — 偏好设置 — 启动自定义格式设置工具，此时控制台的打印结果就很明显了。
