用hook处理组件内定时器

2022-01-01

vue 项目使用 hook 处理组件内定时器的方式。

通常处理组件内定时器的步骤是：

export default{
    mounted(){
        this.timer = setInterval（() =>{
            // do something
        },1000）
    },
    beforeDestory(){
        clearInterval(this.timer);
    }
}

使用 hook 可以更好的处理

export default{
    mounted(){
        const timer = setInterval(() => {
            // do something
        },1000);
        this.$once('hook:beforeDestory',()=> clearInterval(timer);)
    }
}

适用场景:

处理可视化图表的时候，我们需要在数据渲染到页面之前让页面有 loading 动画，mounted 挂载之后停止 loading，beforeUpdate 时开始 loading，updated 之后停止 loading。

使用 hook 处理:

<v-chart>
  @hook:mounted="loading = false" @hook:beforeUpdated="loading = true"
  @hook:updated="loading = false" :data="data"
</v-chart>
